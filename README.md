# Predictive Maintenance System using Hybrid Machine Learning and SHAP

## 📌 Project Overview
This project presents a SHAP-Based Hybrid Machine Learning Approach for Predictive Maintenance of Industrial Equipment. The system predicts machine failures using industrial sensor data and provides explainable insights using SHAP (SHapley Additive Explanations).

The model combines:

XGBoost for high-performance predictive learning
Logistic Regression for interpretable decision-making
SHAP Explainability for feature-level insights

The system also includes an interactive dashboard for real-time prediction and visualization.
